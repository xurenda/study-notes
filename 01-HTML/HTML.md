# HTML

## 1.基础

- 后缀：`.html` 或 `.htm`

- 格式

  ```html
  <标签 属性="值">内容</标签>
  
  <!-- 自闭合标签 -->
  <标签 属性="值" />
  ```

- 基本结构

  ```html
  <!DOCTYPE html>
  <html>
      <head>
          <meta charset="UTF-8">
          <title>页面标题</title>
      </head>
      <body>
          ...
      </body>
  </html>
  ```

## 3.属性

| 属性  | 描述                                                         |
| ----- | ------------------------------------------------------------ |
| class | 为html元素定义一个或多个类名（classname）(类名从样式文件引入) |
| id    | 定义元素的唯一id                                             |
| style | 规定元素的行内样式（inline style）                           |
| title | 描述了元素的额外信息 (作为工具条使用)                        |
